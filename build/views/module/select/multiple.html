接口说明：

多选时，将type设置成功multiple，其中有两种数据来源方式，一种是data静态数据、一种是ajax，选择完表单要提交是直接提交target这个隐藏域的。

应用代码：

/*html 代码*/
<div class="fn-clear">
	<input type="hidden" name="multiple" id="multiple" />
</div>

/*js 代码*/
var select = nic.ui.select2({
	target:'#multiple',
	type:'multiple',
	//data:data, 静态数据
	ajax:{
		url: 'http://127.0.0.1/nic/doc/tmp/select.php',
		data:{
			index:'1'
		}
	}
});
