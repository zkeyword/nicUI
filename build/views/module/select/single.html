接口说明：

该功能是模拟select的形式，html部分的select相关的操作还是跟原生的一样，使用时，直接将接口type设置single即可。

应用代码：

/*html 代码*/
<select id="select" name="select" class="ui-select">
    <option value="0">选项1</option>
    <option value="1">选项2</option>
    <option value="2" selected>选项3</option>
    <option value="3">选项4</option>
</select>

/*js 代码*/
var select = nic.ui.select2({
	target:'select',
	type:'single'
});
